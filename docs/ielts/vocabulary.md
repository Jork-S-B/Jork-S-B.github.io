分发，分配

| 词汇         | 适用场景          | 例句                                              |
|------------|---------------|-------------------------------------------------|
| **dispense**   | 按需/按量分发药品、建议等 | The machine dispenses medication automatically. |
| **distribute** | 广泛分配物资、信息     | We distributed supplies to all villages.        |
| **allocate**   | 系统性分派资源、预算等   | The manager allocated tasks to each team.       |
| **assign**     | 分配任务/职责       | Teachers assign homework.                       |
| apportion  | 按比例分配（正式）     | Costs were apportioned fairly.                  |
| dole out   | 少量分发（常含贬义）    | He doled out minimal rewards.                   |
| **ration**     | 限量配给（稀缺资源）    | Water was rationed during drought.              |
| mete out   | 施以惩罚/奖励（正式）   | The judge meted out a harsh sentence.           |
| circulate  | 分发信息/文件       | Please circulate the memo.                      |
